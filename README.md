## gRPC によるクライエントとサーバ通信

主な通信の方式
1. Unary RPC (1req 1 res)
2. サーバーストリーミング(1req 多res)
3. クライアントストリーミング(多req 1res)
4. 双方向ストリーミング (多req 多res)


##   その他技術

#### Interceptor

#### ssl通信
